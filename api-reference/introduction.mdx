## Welcome

There are two ways to build API documentation: [OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup) and [MDX components](https://mintlify.com/docs/api-playground/mdx/configuration). For the starter kit, we are using the OpenAPI specification.

## Authentication

All API endpoints are authenticated using Bearer tokens and picked up from the specification file.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```

## Base URL

All API requests should be made to:

```bash
http://sandbox.mintlify.com
```

2. **Make your first request** - Try the [Get Plants](/api-reference/endpoint/get) endpoint

## OpenAPI Integration

Mintlify can automatically generate API documentation from your OpenAPI specification file. Simply reference the OpenAPI endpoint in your page frontmatter:

```yaml
---
title: 'Get Plants'
openapi: 'GET /plants'
---
```

This will automatically generate:
- Parameter documentation
- Request body schemas
- Response examples
- Interactive API playground
- Code examples in multiple languages
