Mintlify uses MDX, which combines standard Markdown with powerful React components. This allows you to write clean, readable content while adding interactive elements where needed.

## Basic Markdown Syntax

### Headings

Use `##` for section headers and `###` for subsections:

```md
## Section Header

### Subsection Header
```

<Tip>
Each heading creates an anchor link and appears in the table of contents on the right side of the page.
</Tip>

### Text Formatting

Format text using simple markdown syntax:

| Style         | Syntax            | Result          |
| ------------- | ----------------- | --------------- |
| Bold          | `**bold**`        | **bold**        |
| Italic        | `_italic_`        | _italic_        |
| Strikethrough | `~strikethrough~` | ~strikethrough~ |

You can combine these styles. For example, `**_bold and italic_**` produces **_bold and italic_** text.

For superscript and subscript, use HTML tags:

| Style       | Syntax                   | Result                 |
| ----------- | ------------------------ | ---------------------- |
| Superscript | `<sup>superscript</sup>` | <sup>superscript</sup> |
| Subscript   | `<sub>subscript</sub>`   | <sub>subscript</sub>   |

### Links

Create links by wrapping text in `[text](url)` format:

```md
[Link to Google](https://google.com)
```

<Note>
For internal links, use root-relative paths like `/essentials/code` instead of relative paths like `../code`. This allows Mintlify to optimize page loading.
</Note>

### Blockquotes

Create blockquotes with the `>` character:

```md
> Single line blockquote
```

For multiline blockquotes, use `>` on each line:

```md
> This is the first line of the blockquote.
>
> This is the second paragraph in the same blockquote.
```

### Lists

Create unordered lists with `-` or `*`:

```md
- First item
- Second item
- Third item
```

Create ordered lists with numbers:

```md
1. First step
2. Second step
3. Third step
```

## MDX Components

Mintlify provides special components to enhance your documentation.

### Callout Components

Highlight important information with callout components:

<Note>
This is a note. Use it for helpful information that supports the main content.
</Note>

```mdx
<Note>
This is a note. Use it for helpful information.
</Note>
```

<Tip>
This is a tip. Use it for best practices and helpful suggestions.
</Tip>

```mdx
<Tip>
This is a tip. Use it for best practices.
</Tip>
```

<Warning>
This is a warning. Use it to alert users about potential issues.
</Warning>

```mdx
<Warning>
This is a warning. Use it to alert users about potential issues.
</Warning>
```

<Info>
This is an info callout. Use it for prerequisite information or context.
</Info>

```mdx
<Info>
This is an info callout. Use it for prerequisites or context.
</Info>
```

### Card Components

Organize content with cards. Use `<CardGroup>` to display multiple cards:

<CardGroup cols={2}>
  <Card title="First Feature" icon="rocket" href="/quickstart">
    Description of the first feature
  </Card>
  <Card title="Second Feature" icon="code" href="/essentials/code">
    Description of the second feature
  </Card>
</CardGroup>

```mdx
<CardGroup cols={2}>
  <Card title="First Feature" icon="rocket" href="/quickstart">
    Description of the first feature
  </Card>
  <Card title="Second Feature" icon="code" href="/essentials/code">
    Description of the second feature
  </Card>
</CardGroup>
```

### Accordion Components

Create expandable sections with accordions:

<AccordionGroup>
  <Accordion title="What is Mintlify?" icon="question">
    Mintlify is a modern documentation platform that helps you create beautiful, functional docs.
  </Accordion>
  <Accordion title="How do I get started?" icon="rocket">
    Follow our quickstart guide to set up your docs in minutes.
  </Accordion>
</AccordionGroup>

```mdx
<AccordionGroup>
  <Accordion title="What is Mintlify?" icon="question">
    Content goes here.
  </Accordion>
  <Accordion title="How do I get started?" icon="rocket">
    More content here.
  </Accordion>
</AccordionGroup>
```

### Steps Component

Create step-by-step instructions:

<Steps>
  <Step title="First Step">
    Complete the first task.
  </Step>
  <Step title="Second Step">
    Complete the second task.
  </Step>
</Steps>

```mdx
<Steps>
  <Step title="First Step">
    Complete the first task.
  </Step>
  <Step title="Second Step">
    Complete the second task.
  </Step>
</Steps>
```

### LaTeX Support

Display mathematical formulas with the `<Latex>` component:

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```mdx
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
